# Генерация уравнений
По условиям задачи необходимо использовать более 100 мб на 1 вычислительный узел.
Пусть n - число узлов, k - число уравнений, а l - длина выбранного типа данных в байтах.
Тогда 

k * (k + 1) * l > 100 * n

l * k^2 + l * k - 100 * n > 0

Следовательно

k > 1/2 * (sqrt((l + 400 * n) / l) - 1)
